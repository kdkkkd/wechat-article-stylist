# Sample Prompts

## 通用写作

```text
使用 $wechat-article-stylist，写一篇关于“为什么厉害的人都在刻意减少无效社交”的公众号文章，风格克制、有观点，适合职场成长类账号。
```

## 干货型

```text
Use $wechat-article-stylist to draft a polished WeChat article explaining 5 practical ways to improve writing clarity for knowledge creators.
```

## 故事型

```text
使用 $wechat-article-stylist，把这段创业经历改写成公众号故事稿，要求开头有冲突感，正文有层次，结尾有共鸣和互动引导。
```

## 品牌号

```text
使用 $wechat-article-stylist，写一篇适合品牌公众号发布的文章，主题是“为什么我们坚持做更慢一点的产品”，语气真诚、克制，不要太像广告。
```

## 转化型

```text
使用 $wechat-article-stylist，为一次写作训练营活动写公众号预热文章，要求有标题备选、摘要、正文排版稿和文末转化引导。
```

## 改稿优化

```text
使用 $wechat-article-stylist，把我现有的草稿优化成更适合公众号发布的版本，重点优化标题、开头、小标题和排版节奏。
```


## AI 知识博主

```text
使用 $wechat-article-stylist，帮我细化一个“AI 知识博主公众号”风格：要求专业、克制、信息密度高，但不要太像说明书；可以自然提到 OpenClaw、OpenAI、工作流和 Agent 的协作方式。
```

## OpenClaw 场景

```text
Use $wechat-article-stylist to write a WeChat article for an AI creator account about how to combine OpenClaw with mainstream models in a practical content workflow. Include title options, a subtitle, summary, body layout, and a short section on common mistakes.
```
